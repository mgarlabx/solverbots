Notification componente reutilizável

Sistema de notificações com tipos (success, error, warning, info) e duração configurável.

Tipos de Notificação

Clique num botão para ver a notificação correspondente.

Success

Error

Warning

Info

Duração Personalizada

Controle por quanto tempo a notificação fica visível (em milissegundos).

Tipo Padrão

Quando nenhum tipo é especificado, usa info por defeito.

Mensagens Longas

Comportamento com textos mais extensos.

Como usar

// Notificação de sucesso (duração padrão: 2s) Notification.show('Guardado com sucesso!', 'success'); // Notificação de erro Notification.show('Ocorreu um erro.', 'error'); // Notificação de aviso Notification.show('Verifique os dados.', 'warning'); // Notificação informativa Notification.show('Sessão iniciada.', 'info'); // Com duração personalizada (5 segundos) Notification.show('Esta fica 5s visível.', 'success', 5000); // Sem tipo (usa 'info' por defeito) Notification.show('Mensagem simples.');
